=================================================
[JavaScript Operators](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Expressions_and_Operators)
=================================================

-------------------------------------------------
[Equality Operators](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Expressions_and_Operators#comparison_operators)
-------------------------------------------------
==     # Equal (with type coercion)
!=     # Not equal (with type coercion)
===    # Strict equal (no type coercion)
!==    # Strict not equal


-------------------------------------------------
[Relational Operators](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Expressions_and_Operators#comparison_operators)
-------------------------------------------------
>      # Greater than
<      # Less than
>=     # Greater than or equal to
<=     # Less than or equal to
in     # Property exists in object
instanceof # Prototype chain check


-------------------------------------------------
[Assignment Operators](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Expressions_and_Operators#assignment_operators)
-------------------------------------------------
=      # Assignment
+=     # Add and assign
-=     # Subtract and assign
*=     # Multiply and assign
/=     # Divide and assign
%=     # Modulo and assign
**=    # Exponentiation and assign
&&=    # Logical AND assignment
||=    # Logical OR assignment
??=    # Nullish coalescing assignment


-------------------------------------------------
[Arithmetic Operators](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Expressions_and_Operators#arithmetic_operators)
-------------------------------------------------
+      # Addition / String concatenation
-      # Subtraction / Unary minus
*      # Multiplication
/      # Division
%      # Remainder
**     # Exponentiation
++     # Increment
--     # Decrement


-------------------------------------------------
[Logical Operators](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Expressions_and_Operators#logical_operators)
-------------------------------------------------
&&     # Logical AND
||     # Logical OR
!      # Logical NOT


-------------------------------------------------
[Nullish & Optional Operators](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Expressions_and_Operators#logical_operators)
-------------------------------------------------
??     # Nullish coalescing (null or undefined)
?.     # Optional chaining


-------------------------------------------------
[Bitwise Operators](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Expressions_and_Operators#bitwise_operators)
-------------------------------------------------
&      # Bitwise AND
|      # Bitwise OR
^      # Bitwise XOR
~      # Bitwise NOT
<<     # Left shift
>>     # Sign-propagating right shift
>>>    # Zero-fill right shift


-------------------------------------------------
[Conditional Operator](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Expressions_and_Operators#conditional_operator)
-------------------------------------------------
?:     # Ternary conditional operator


-------------------------------------------------
[Type & Unary Operators](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Expressions_and_Operators#unary_operators)
-------------------------------------------------
typeof # Type of operand
delete # Delete object property
void   # Discard expression result
new    # Object instantiation
await  # Await Promise (async context)


-------------------------------------------------
[Access & Invocation Operators](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Expressions_and_Operators#property_accessors)
-------------------------------------------------
.      # Property access
[]     # Property access (dynamic)
()     # Function invocation
